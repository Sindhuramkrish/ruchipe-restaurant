# Restaurant Management Web App

## Overview
This is a **responsive full-stack web application** built using Flask, designed to simplify restaurant order management.  
It helps restaurant staff and owners — even with low digital literacy — take customer orders quickly using a visual, image-based interface, manage menus, track sales, and generate bills.  

The app is deployed online and uses cloud services for database and image storage.

---

## Tech Stack
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS, Jinja templates  
- **Database:** Cloud-hosted SQL database using SQLAlchemy  
- **Image Storage:** Cloudinary  
- **Deployment:** Render  

---

## Features
- Fully responsive UI (mobile, tablet, desktop)  
- Image-based menu for fast order placement  
- View order history with date and time  
- Monthly & yearly sales analysis  
- Item-wise sales tracking (most sold items)  
- Bill printing support when connected to a printer  
- Safe order cancellation (canceled orders do not affect DB)  
- Dynamic menu management (add/update items)
